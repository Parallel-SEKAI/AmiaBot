# 指令参考

本文档详细列出了 AmiaBot 支持的各项指令、调用方式和示例。

## 通用指令

| 功能         | 指令/触发方式           | 描述                   | 示例                      |
| :----------- | :---------------------- | :--------------------- | :------------------------ |
| **AI 聊天**  | `amia <你的问题>`       | 与机器人进行智能对话。 | `amia 你叫什么名字？`     |
| **显示帮助** | `/help`                 | 显示帮助信息。         | `/help`                   |
| **功能控制** | `@机器人 /bot feat ...` | 开启或关闭机器人功能。 | `@AmiaBot /bot feat list` |

## 信息查询

| 功能              | 指令/触发方式              | 描述                               | 示例                                  |
| :---------------- | :------------------------- | :--------------------------------- | :------------------------------------ |
| **Bilibili 解析** | 发送B站链接或号码          | 自动解析视频信息并发送卡片。       | `BV1XY411A7c2`                        |
| **GitHub 解析**   | 发送 GitHub 链接           | 自动解析仓库、用户、Issue 等信息。 | `https://github.com/microsoft/vscode` |
| **Gemini 问答**   | `/gemini <问题>`           | 使用 Gemini 模型进行问答。         | `/gemini 解释一下AI`                  |
| **查询信息**      | `/query` 或 `/query @用户` | 查询自己或他人的群内信息。         | `/query @张三`                        |
| **查询群信息**    | `/group`                   | 查询当前群聊的详细信息。           | `/group`                              |
| **消息统计**      | `消息统计`                 | AI 分析近期群聊记录并生成报告。    | `消息统计`                            |

## 音乐功能 (网易云)

| 功能           | 指令/触发方式                   | 描述                      | 示例                       |
| :------------- | :------------------------------ | :------------------------ | :------------------------- |
| **搜索歌曲**   | `/search <歌名>`                | 搜索歌曲并返回结果列表。  | `/search Lemon`            |
| **播放歌曲**   | 回复搜索结果 `/play <序号>`     | 播放指定序号的歌曲。      | 回复搜索结果 `/play 1`     |
| **下载歌曲**   | 回复搜索结果 `/download <序号>` | 发送指定序号的歌曲文件。  | 回复搜索结果 `/download 1` |
| **Emoji 歌词** | `/emojilyric <歌名>`            | 将歌词转换为 Emoji 形式。 | `/emojilyric Lemon`        |

## 游戏与娱乐

| 功能          | 指令/触发方式         | 描述                               | 示例              |
| :------------ | :-------------------- | :--------------------------------- | :---------------- |
| **戳一戳**    | 在群里戳机器人        | 触发随机彩蛋互动。                 | (戳机器人)        |
| **随机漫画**  | `/comic`              | 发送一张随机的 PJSK 官方四格漫画。 | `/comic`          |
| **投骰子**    | `.r` 或 `.r[A]d[B]`   | A为骰子数，B为面数。               | `.r2d6`           |
| **PJSK 贴纸** | `/pjsk <角色> <文字>` | 生成 PJSK 风格的表情包。           | `/pjsk miku 你好` |
| **猜卡面**    | `猜卡面 [难度]`       | 开始猜 PJSK 游戏卡面。             | `猜卡面 ex`       |
| **猜活动**    | `猜活动 [难度]`       | 开始猜 PJSK 游戏活动。             | `猜活动`          |
| **听歌识曲**  | `听歌识曲 [难度]`     | 开始 PJSK 听歌识曲游戏。           | `听歌识曲 ez`     |

### 游戏难度说明

在猜谜游戏中，你可以通过附加难度关键词来调整挑战等级：

- `ez`: 简单
- `no`: 普通
- `hd`: 困难
- `ex`: 专家
- `ma`: 大师
- `apd`: 噩梦 (最高难度)

例如，`猜卡面 ma` 会开始一局大师难度的猜卡面游戏。
